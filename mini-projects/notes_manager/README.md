# Notes Manager (Python)

A simple command-line Notes Manager built in Python.  
It lets you add notes, view them, search through them, and clear them when needed.  
All notes are saved in a JSON file with timestamps and auto-generated IDs.

## Features
- Add new notes  
- View all saved notes  
- Search notes by keyword  
- Clear all notes  
- JSON-based storage  
- Basic error handling  

## Files
- `app.py` — main program  
- `notes.json` — stores all notes  
- `README.md` — project documentation

python3 app.py
